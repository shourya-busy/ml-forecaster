Backtest folds (how train/test was split)

Each algorithm above was evaluated with walk-forward cross-validation: the historical series is split into K expanding-window folds; each fold trains on everything before the cutoff and predicts the held-out tail. The "Avg score" columns above are the average across these folds. Once the backtest finishes, the winning model is refit on the full history before producing the production forecast, so deployed models use every data point.
